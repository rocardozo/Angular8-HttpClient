Trabajando con HttpClient y la api 'https://jsonplaceholder.typicode.com'

Pasos a seguir:

  • Crear un servicio con el comando ng g s (nombre del servicio)
  • Dentro del archivo nombre del servicio.service.ts importar HttpClient desde '@angular/common/http';
  • Tambien inyectar en el constructor una variable privada llamda http de tipo HttpClient
  • En el app.module.ts importar el modulo HttpClientModule desde @angular/common/http y el servicio, luego llamar el servicio en providers
  • Por último el al componet.ts a usar el servicio debo importarlo e inyectarlo en el constructor
